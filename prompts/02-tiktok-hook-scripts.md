## TIKTOK OPTIMIZATION NOTES

**Caption Template:**
```
[First line = hook text — same as your verbal hook]
[Second line = what they'll learn/get]
[Third line = CTA]

[Hashtags]
```

**Algorithm Triggers Built Into These Scripts:**
- Watch time: hooks designed to create loops (viewers rewatch)
- Comments: each script has a question or controversy that drives replies
- Saves: variation 1 has a "steal this" or tip format for saves
- Shares: transformation story is most shareable format

**Repurpose Map:**
- YouTube: expand variation 2 (transformation) into 10-minute video
- Instagram Reels: use variation 1 with music (no voice-over needed)
- LinkedIn: strip to text post using variation 3 argument
- Email: use transformation story as subscriber win case study
